# Política de seguridad

Este repositorio contiene una demostración sanitizada para fines educativos y
de portafolio.

## Información que no debe publicarse

- Tokens o claves API.
- Credenciales OAuth.
- Archivos `.env`.
- Documentos reales.
- NIT o identificadores reales.
- URLs privadas.
- IDs de hojas de cálculo.
- Datos de ejecuciones de producción.

## Reporte de vulnerabilidades

No publiques credenciales o datos privados.

Reporta cualquier problema de seguridad de forma privada al mantenedor del
repositorio.

## Alcance

La versión soportada es la última versión publicada en GitHub Releases.
